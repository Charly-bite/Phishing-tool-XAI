Como ejecutar:

1.copiar los datos del dataset en la carpeta "data"
2.Ejecutar "requirements.sh"
3.Ejecutar entorno virtual: .\venv\Scripts\activate
4.Formato del script para ejecutar: python main_script.py --data-dir /path/to/your/csv/data --output-dir /path/to/save/results
Ejemplo de como debe verse la ruta: python main_script.py --data-dir C:\Users\usuario\Desktop\Phishing\data --output-dir "C:\Users\usuario\Desktop\Phishingv\saved data"
5.Una vez terminado el entrenamiento ejecutar python app.py para abrir el front end
